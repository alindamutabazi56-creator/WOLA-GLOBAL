## Typography

## Spacing and radius
- Container max width: 1280px
- Base radius: 1.25rem (`xl2`)
- Component spacing: 0.75rem / 1rem / 1.5rem / 2.5rem

## Component usage
- Cards: rounded, soft shadow, concise title+meta+CTA
- Buttons: pill-shaped, high-contrast states
- Filter chips: bordered, keyboard-focus-visible
- Forms: labelled controls, honeypot anti-spam + server validation
