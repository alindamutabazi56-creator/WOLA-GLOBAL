# WOLA Brand & Style Guide

## Colour tokens
- `brand.emerald` `#0F3D2E`
- `brand.gold` `#C6A25D`
- `brand.cream` `#F8F4EB`
- `brand.charcoal` `#111111`

## Typography
- Heading: Playfair Display (`font-serif`)
- Body/UI: Inter (`font-sans`)

## Spacing & surfaces
- Section spacing: `space-y-8` or `space-y-14`
- Cards: rounded (`rounded-xl2`), soft shadow (`shadow-soft`)

## Component usage
- Navigation: sticky, compact, keyboard-focus visible.
- Cards: event and post cards with media + concise CTA.
- Forms: server actions, honeypot field, validation and rate limiting.
- Data sections: metrics, testimonials, and partner chips.
