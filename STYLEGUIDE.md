# WOLA Style Guide

## Colour tokens
- `emerald` #0F3D2E
- `ink` #0F0F0F
- `cream` #F7F2E8
- `gold` #C5A46D
- `muted` #617B72

## Typography
- Heading: Playfair Display (serif)
- Body: Inter (sans-serif)
- Scale: 14 / 16 / 18 / 24 / 32 / 48

## Component usage
- Buttons: `btn-primary` for main CTA, `btn-secondary` for alternate CTA.
- Cards: use `.card` for all elevated content blocks.
- Spacing: section-level 48px+ with consistent rhythm.
- Motion: subtle fade/slide transitions only.
